== Description ==

Combo WP Rewrite Slugs is an easy to use plugin which provides functionality to rewrite your WordPress website URLs with not just ease but also with 100% accuracy.

= What Combo WP Rewrite Slugs Will Does and Features =

* Rewrite Custom Post Types
* Rewrite Categories and Taxonomies base
* Rewrite WordPress Users base slug by user Roles
* Completely Ajax base
* Well Managed code
* Easy to use
* More features coming soon....

== Installation ==

1. Upload the entire `combo-wp-rewrite-slugs` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Rewrite Slugs' menu under Tools menu in your WordPress admin panel.
